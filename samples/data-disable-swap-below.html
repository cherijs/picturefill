Picturefill

data-disable-swap-below

Prevents Picturefill from applying a div[data-src] tag below the one that is currently applied.

Instructions

Start with your browser window large and then scale it small, you will notice the images will swap out. Then scale your browser window large again, you will notice that it does not swap out.

Code sample

<div data-picture data-alt="Picturefill test" data-disable-swap-below> <div data-src="img/default.png"></div> <div data-src="img/320x90.png" data-media="(min-width: 200px)"></div> <div data-src="img/480x135.png" data-media="(min-width: 321px)"></div> <div data-src="img/640x180.png" data-media="(min-width: 481px)"></div> <div data-src="img/1024x228.png" data-media="(min-width: 641px)"></div> <div data-src="img/1440x405.png" data-media="(min-width: 1025px)"></div> <div data-src="img/1920x540.png" data-media="(min-width: 1441px)"></div> <noscript><img src="img/noscript.png" alt="Picturefill test" /></noscript> </div>
